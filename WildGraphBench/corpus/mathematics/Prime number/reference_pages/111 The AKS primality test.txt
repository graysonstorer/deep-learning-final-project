The AKS primality test
======================

11 August, 2009 in [expository](https://terrytao.wordpress.com/category/expository/), [math.NT](https://terrytao.wordpress.com/category/mathematics/mathnt/), [polymath](https://terrytao.wordpress.com/category/question/polymath/) | Tags: [AKS primality test](https://terrytao.wordpress.com/tag/aks-primality-test/), [finding primes](https://terrytao.wordpress.com/tag/finding-primes/)

The [Agrawal-Kayal-Saxena (AKS) primality test](http://en.wikipedia.org/wiki/AKS_primality_test), discovered in 2002, is the first provably deterministic algorithm to determine the primality of a given number with a run time which is guaranteed to be polynomial in the number of digits, thus, given a large number ![Image 233: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), the algorithm will correctly determine whether that number is prime or not in time ![Image 234: {O(\log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7BO%28%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0). (Many previous primality testing algorithms existed, but they were either probabilistic in nature, had a running time slower than polynomial, or the correctness could not be guaranteed without additional hypotheses such as GRH.)

The AKS test is of some relevance to the polymath project “[Finding primes](http://michaelnielsen.org/polymath1/index.php?title=Finding_primes)“, so I thought I would sketch the details of the test (and the proof that it works) here. (Of course, full details can be found in the [original paper](http://www.cse.iitk.ac.in/users/manindra/algebra/primality_v6.pdf), which is nine pages in length and almost entirely elementary in nature.) It relies on polynomial identities that are true modulo ![Image 235: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) when ![Image 236: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is prime, but cannot hold for ![Image 237: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) non-prime as they would generate a large number of additional polynomial identities, eventually violating the [factor theorem](http://en.wikipedia.org/wiki/Factor_theorem) (which asserts that a polynomial identity of degree at most ![Image 238: {d}](https://s0.wp.com/latex.php?latex=%7Bd%7D&bg=ffffff&fg=000000&s=0) can be obeyed by at most ![Image 239: {d}](https://s0.wp.com/latex.php?latex=%7Bd%7D&bg=ffffff&fg=000000&s=0) values of the unknown).

Our starting point is [Fermat’s little theorem](http://en.wikipedia.org/wiki/Fermat%27s_little_theorem), which asserts that [](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)

[![Image 240: \displaystyle a^p = a \hbox{ mod } p \ \ \ \ \ (1)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++a%5Ep+%3D+a+%5Chbox%7B+mod+%7D+p+%5C+%5C+%5C+%5C+%5C+%281%29&bg=ffffff&fg=000000&s=0)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)
[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/) for every prime ![Image 241: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) and every ![Image 242: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0). This theorem suggests an obvious primality test: to test whether a number ![Image 243: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is prime, pick a few values of ![Image 244: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) and see whether ![Image 245: {a^n = a \hbox{ mod } n}](https://s0.wp.com/latex.php?latex=%7Ba%5En+%3D+a+%5Chbox%7B+mod+%7D+n%7D&bg=ffffff&fg=000000&s=0). (Note that ![Image 246: {a^n}](https://s0.wp.com/latex.php?latex=%7Ba%5En%7D&bg=ffffff&fg=000000&s=0) can be computed in time ![Image 247: {O(\log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7BO%28%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0) for any fixed ![Image 248: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) by expressing ![Image 249: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) in binary, and repeatedly squaring ![Image 250: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0).) If the statement ![Image 251: {a^n = a \hbox{ mod } n}](https://s0.wp.com/latex.php?latex=%7Ba%5En+%3D+a+%5Chbox%7B+mod+%7D+n%7D&bg=ffffff&fg=000000&s=0) fails for some ![Image 252: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0), then ![Image 253: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) would be composite. Unfortunately, the converse is not true: there exist non-prime numbers ![Image 254: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), known as [Carmichael numbers](http://en.wikipedia.org/wiki/Carmichael_number), for which ![Image 255: {a^n = a \hbox{ mod } n}](https://s0.wp.com/latex.php?latex=%7Ba%5En+%3D+a+%5Chbox%7B+mod+%7D+n%7D&bg=ffffff&fg=000000&s=0) for all ![Image 256: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) coprime to ![Image 257: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) (![Image 258: {561}](https://s0.wp.com/latex.php?latex=%7B561%7D&bg=ffffff&fg=000000&s=0) is the first example). So Fermat’s little theorem cannot be used, by itself, to establish primality for general ![Image 259: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), because it is too weak to eliminate all non-prime numbers. (The situation improves though for more special types of ![Image 260: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), such as Mersenne numbers; see [my earlier post on the Lucas-Lehmer test](https://terrytao.wordpress.com/2008/10/02/the-lucas-lehmer-test-for-mersenne-primes/) for more discussion.)

However, there is a stronger version of Fermat’s little theorem which does eliminate all non-prime numbers. Specifically, if ![Image 261: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) is prime and ![Image 262: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) is arbitrary, then one has the polynomial identity [](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)

[![Image 263: \displaystyle (X+a)^p = X^p + a \hbox{ mod } p \ \ \ \ \ (2)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%28X%2Ba%29%5Ep+%3D+X%5Ep+%2B+a+%5Chbox%7B+mod+%7D+p+%5C+%5C+%5C+%5C+%5C+%282%29&bg=ffffff&fg=000000&s=0)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)
[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/) where ![Image 264: {X}](https://s0.wp.com/latex.php?latex=%7BX%7D&bg=ffffff&fg=000000&s=0) is an indeterminate variable. (More formally, we have the identity ![Image 265: {(X+a)^p=X^p+a}](https://s0.wp.com/latex.php?latex=%7B%28X%2Ba%29%5Ep%3DX%5Ep%2Ba%7D&bg=ffffff&fg=000000&s=0) in the ring ![Image 266: {F_p[X]}](https://s0.wp.com/latex.php?latex=%7BF_p%5BX%5D%7D&bg=ffffff&fg=000000&s=0) of polynomials of one variable ![Image 267: {X}](https://s0.wp.com/latex.php?latex=%7BX%7D&bg=ffffff&fg=000000&s=0) over the finite field ![Image 268: {F_p}](https://s0.wp.com/latex.php?latex=%7BF_p%7D&bg=ffffff&fg=000000&s=0) of ![Image 269: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) elements.) This identity (a manifestation of the [Frobenius endomorphism](http://en.wikipedia.org/wiki/Frobenius_endomorphism)) clearly implies [(1)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#flt) by setting ![Image 270: {X=0}](https://s0.wp.com/latex.php?latex=%7BX%3D0%7D&bg=ffffff&fg=000000&s=0); conversely, one can easily deduce [(2)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt) from [(1)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#flt) by expanding out ![Image 271: {(X+a)^p}](https://s0.wp.com/latex.php?latex=%7B%28X%2Ba%29%5Ep%7D&bg=ffffff&fg=000000&s=0) using the [binomial theorem](http://en.wikipedia.org/wiki/Binomial_theorem) and the observation that the binomial coefficients ![Image 272: {\binom{p}{i} = \frac{p \cdot \ldots \cdot (p-i+1)}{i!}}](https://s0.wp.com/latex.php?latex=%7B%5Cbinom%7Bp%7D%7Bi%7D+%3D+%5Cfrac%7Bp+%5Ccdot+%5Cldots+%5Ccdot+%28p-i%2B1%29%7D%7Bi%21%7D%7D&bg=ffffff&fg=000000&s=0) are divisible by ![Image 273: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) for all ![Image 274: {1 \leq i < p}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+i+%3C+p%7D&bg=ffffff&fg=000000&s=0). Conversely, if [](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)

[![Image 275: \displaystyle (X+a)^n = X^n + a \hbox{ mod } n \ \ \ \ \ (3)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%28X%2Ba%29%5En+%3D+X%5En+%2B+a+%5Chbox%7B+mod+%7D+n+%5C+%5C+%5C+%5C+%5C+%283%29&bg=ffffff&fg=000000&s=0)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)
[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/) (i.e. ![Image 276: {(X+a)^n = X^n+a}](https://s0.wp.com/latex.php?latex=%7B%28X%2Ba%29%5En+%3D+X%5En%2Ba%7D&bg=ffffff&fg=000000&s=0) in ![Image 277: {({\mathbb Z}/n{\mathbb Z})[X]}](https://s0.wp.com/latex.php?latex=%7B%28%7B%5Cmathbb+Z%7D%2Fn%7B%5Cmathbb+Z%7D%29%5BX%5D%7D&bg=ffffff&fg=000000&s=0)) for some ![Image 278: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) coprime to ![Image 279: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), then by comparing coefficients using the binomial theorem we see that ![Image 280: {\binom{n}{i}}](https://s0.wp.com/latex.php?latex=%7B%5Cbinom%7Bn%7D%7Bi%7D%7D&bg=ffffff&fg=000000&s=0) is divisible by ![Image 281: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) for all ![Image 282: {1 \leq i < n}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+i+%3C+n%7D&bg=ffffff&fg=000000&s=0). But if ![Image 283: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is divisible by some smaller prime ![Image 284: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0), then by setting ![Image 285: {i}](https://s0.wp.com/latex.php?latex=%7Bi%7D&bg=ffffff&fg=000000&s=0) equal to the largest power of ![Image 286: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) that divides ![Image 287: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), one sees that ![Image 288: {\binom{n}{i}}](https://s0.wp.com/latex.php?latex=%7B%5Cbinom%7Bn%7D%7Bi%7D%7D&bg=ffffff&fg=000000&s=0) is not divisible by enough powers of ![Image 289: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) to be divisible by ![Image 290: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), a contradiction. Thus one can use [(3)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-2) (for a single value of ![Image 291: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) coprime to ![Image 292: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0)) to decide whether ![Image 293: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is prime or not.

Unfortunately, this algorithm, while deterministic, is not polynomial-time, because the polynomial ![Image 294: {(X+a)^n}](https://s0.wp.com/latex.php?latex=%7B%28X%2Ba%29%5En%7D&bg=ffffff&fg=000000&s=0) has ![Image 295: {n+1}](https://s0.wp.com/latex.php?latex=%7Bn%2B1%7D&bg=ffffff&fg=000000&s=0) coefficients and will therefore take at least ![Image 296: {O(n)}](https://s0.wp.com/latex.php?latex=%7BO%28n%29%7D&bg=ffffff&fg=000000&s=0) time to compute. However, one can speed up the process by descending to a quotient ring of ![Image 297: {({\mathbb Z}/n{\mathbb Z})[X]}](https://s0.wp.com/latex.php?latex=%7B%28%7B%5Cmathbb+Z%7D%2Fn%7B%5Cmathbb+Z%7D%29%5BX%5D%7D&bg=ffffff&fg=000000&s=0), such as ![Image 298: {{\mathbb Z}/n{\mathbb Z}[X]/(X^r-1)}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathbb+Z%7D%2Fn%7B%5Cmathbb+Z%7D%5BX%5D%2F%28X%5Er-1%29%7D&bg=ffffff&fg=000000&s=0) for some ![Image 299: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0). Clearly, if the identity ![Image 300: {(X+a)^n=X^n+a}](https://s0.wp.com/latex.php?latex=%7B%28X%2Ba%29%5En%3DX%5En%2Ba%7D&bg=ffffff&fg=000000&s=0) holds in ![Image 301: {({\mathbb Z}/n{\mathbb Z})[X]}](https://s0.wp.com/latex.php?latex=%7B%28%7B%5Cmathbb+Z%7D%2Fn%7B%5Cmathbb+Z%7D%29%5BX%5D%7D&bg=ffffff&fg=000000&s=0), then it will also hold in ![Image 302: {({\mathbb Z}/n{\mathbb Z})[X]/(X^r-1)}](https://s0.wp.com/latex.php?latex=%7B%28%7B%5Cmathbb+Z%7D%2Fn%7B%5Cmathbb+Z%7D%29%5BX%5D%2F%28X%5Er-1%29%7D&bg=ffffff&fg=000000&s=0), thus [](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)

[![Image 303: \displaystyle (X+a)^n = X^n + a \hbox{ mod } n, X^r-1. \ \ \ \ \ (4)](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%28X%2Ba%29%5En+%3D+X%5En+%2B+a+%5Chbox%7B+mod+%7D+n%2C+X%5Er-1.+%5C+%5C+%5C+%5C+%5C+%284%29&bg=ffffff&fg=000000&s=0)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)
[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/) The point of doing this is that (if ![Image 304: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) is not too large) the left-hand side of [(4)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-3) can now be computed quickly (again by expanding ![Image 305: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) in binary and performing repeated squaring), because all polynomials can be reduced to be of degree less than ![Image 306: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0), rather than being as large as ![Image 307: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0). Indeed, if ![Image 308: {r = O( \log^{O(1)} n )}](https://s0.wp.com/latex.php?latex=%7Br+%3D+O%28+%5Clog%5E%7BO%281%29%7D+n+%29%7D&bg=ffffff&fg=000000&s=0), then one can test [(4)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-3) in time ![Image 309: {O(\log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7BO%28%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0).

We are not done yet, because it could happen that [(4)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-3) holds but [(3)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-2) fails. But we have the following key theorem:

> **Theorem 1 (AKS theorem)**[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/) Suppose that for all ![Image 310: {1 \leq a, r \leq O( \log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+a%2C+r+%5Cleq+O%28+%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0), [(4)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-3) holds, and ![Image 311: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) is coprime to ![Image 312: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0). Then ![Image 313: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is either a prime, or a power of a prime.

Of course, coprimality of ![Image 314: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) and ![Image 315: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) can be quickly tested using the [Euclidean algorithm](http://en.wikipedia.org/wiki/Euclidean_algorithm), and if coprimality fails then ![Image 316: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is of course composite. Also, it is easy to quickly test for the property that ![Image 317: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is a power of an integer (just compute the roots ![Image 318: {n^{1/k}}](https://s0.wp.com/latex.php?latex=%7Bn%5E%7B1%2Fk%7D%7D&bg=ffffff&fg=000000&s=0) for ![Image 319: {1 \leq k \leq \log_2 n}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+k+%5Cleq+%5Clog_2+n%7D&bg=ffffff&fg=000000&s=0)), and such powers are clearly composite. From all this (and [(2)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt), one soon sees that theorem gives rise to a deterministic polynomial-time test for primality. One can optimise the powers of ![Image 320: {\log n}](https://s0.wp.com/latex.php?latex=%7B%5Clog+n%7D&bg=ffffff&fg=000000&s=0) in the bounds for ![Image 321: {a,r}](https://s0.wp.com/latex.php?latex=%7Ba%2Cr%7D&bg=ffffff&fg=000000&s=0) (as is done in the [original paper](http://www.cse.iitk.ac.in/users/manindra/algebra/primality/_v6.pdf)), but we will not do so here to keep the exposition uncluttered.

Actually, we don’t need [(4)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-3) satisfied for all that many exponents ![Image 322: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) to make the theorem work; just one well-chosen ![Image 323: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) will do. More precisely, we have

> **Theorem 2 (AKS theorem, key step)**[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/) Let ![Image 324: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) be coprime to ![Image 325: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), and such that ![Image 326: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) has order greater than ![Image 327: {\log^2_2 n}](https://s0.wp.com/latex.php?latex=%7B%5Clog%5E2_2+n%7D&bg=ffffff&fg=000000&s=0) in the multiplicative group ![Image 328: {({\mathbb Z}/r{\mathbb Z})^\times}](https://s0.wp.com/latex.php?latex=%7B%28%7B%5Cmathbb+Z%7D%2Fr%7B%5Cmathbb+Z%7D%29%5E%5Ctimes%7D&bg=ffffff&fg=000000&s=0) (i.e. the residues ![Image 329: {n^i \hbox{ mod } r}](https://s0.wp.com/latex.php?latex=%7Bn%5Ei+%5Chbox%7B+mod+%7D+r%7D&bg=ffffff&fg=000000&s=0) for ![Image 330: {1 \leq i \leq \log^2 n}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+i+%5Cleq+%5Clog%5E2+n%7D&bg=ffffff&fg=000000&s=0) are distinct). Suppose that for all ![Image 331: {1 \leq a \leq O( r \log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+a+%5Cleq+O%28+r+%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0), [(4)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-3) holds, and ![Image 332: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0) is coprime to ![Image 333: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0). Then ![Image 334: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is either a prime, or a power of a prime.

To find an ![Image 335: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) with the above properties we have

> **Lemma 3 (Existence of good ![Image 336: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0))**[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/) There exists ![Image 337: {r = O(\log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7Br+%3D+O%28%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0) coprime to ![Image 338: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), such that ![Image 339: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) has order greater than ![Image 340: {\log^2_2 n}](https://s0.wp.com/latex.php?latex=%7B%5Clog%5E2_2+n%7D&bg=ffffff&fg=000000&s=0) in ![Image 341: {({\mathbb Z}/r{\mathbb Z})^\times}](https://s0.wp.com/latex.php?latex=%7B%28%7B%5Cmathbb+Z%7D%2Fr%7B%5Cmathbb+Z%7D%29%5E%5Ctimes%7D&bg=ffffff&fg=000000&s=0).

_Proof:_ For each ![Image 342: {1 \leq i \leq \log^2_2 n}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+i+%5Cleq+%5Clog%5E2_2+n%7D&bg=ffffff&fg=000000&s=0), the number ![Image 343: {n^i - 1}](https://s0.wp.com/latex.php?latex=%7Bn%5Ei+-+1%7D&bg=ffffff&fg=000000&s=0) has at most ![Image 344: {O( \log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7BO%28+%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0) prime divisors (by the fundamental theorem of arithmetic). If one picks ![Image 345: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) to be the first prime not equal to any of these prime divisors, one obtains the claim. (One can use a crude version of the prime number theorem to get the upper bound on ![Image 346: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0).) ![Image 347: \Box](https://s0.wp.com/latex.php?latex=%5CBox&bg=ffffff&fg=000000&s=0)

It is clear that Theorem [1](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#main) follows from Theorem [2](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#key) and Lemma [3](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#good), so it suffices now to prove Theorem [2](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#key).

Suppose for contradiction that Theorem [2](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#key) fails. Then ![Image 348: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is divisible by some smaller prime ![Image 349: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0), but is not a power of ![Image 350: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0). Since ![Image 351: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is coprime to all numbers of size ![Image 352: {O(\log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7BO%28%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0) we know that ![Image 353: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) is not of polylogarithmic size, thus we may assume ![Image 354: {p \geq \log^C n}](https://s0.wp.com/latex.php?latex=%7Bp+%5Cgeq+%5Clog%5EC+n%7D&bg=ffffff&fg=000000&s=0) for any fixed ![Image 355: {C}](https://s0.wp.com/latex.php?latex=%7BC%7D&bg=ffffff&fg=000000&s=0). As ![Image 356: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) is coprime to ![Image 357: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0), we see that ![Image 358: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) is not a multiple of ![Image 359: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) (indeed, one should view ![Image 360: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0) as being much larger than ![Image 361: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0)).

Let ![Image 362: {F}](https://s0.wp.com/latex.php?latex=%7BF%7D&bg=ffffff&fg=000000&s=0) be a field extension of ![Image 363: {F_p}](https://s0.wp.com/latex.php?latex=%7BF_p%7D&bg=ffffff&fg=000000&s=0) by a primitive ![Image 364: {r^{th}}](https://s0.wp.com/latex.php?latex=%7Br%5E%7Bth%7D%7D&bg=ffffff&fg=000000&s=0) root of unity ![Image 365: {X}](https://s0.wp.com/latex.php?latex=%7BX%7D&bg=ffffff&fg=000000&s=0), thus ![Image 366: {F = F_p[X]/h(X)}](https://s0.wp.com/latex.php?latex=%7BF+%3D+F_p%5BX%5D%2Fh%28X%29%7D&bg=ffffff&fg=000000&s=0) for some factor ![Image 367: {h(X)}](https://s0.wp.com/latex.php?latex=%7Bh%28X%29%7D&bg=ffffff&fg=000000&s=0) (in ![Image 368: {F_p[X]}](https://s0.wp.com/latex.php?latex=%7BF_p%5BX%5D%7D&bg=ffffff&fg=000000&s=0)) of the ![Image 369: {r^{th}}](https://s0.wp.com/latex.php?latex=%7Br%5E%7Bth%7D%7D&bg=ffffff&fg=000000&s=0)[cyclotomic polynomial](http://en.wikipedia.org/wiki/Cyclotomic_polynomial)![Image 370: {\Phi_r(X)}](https://s0.wp.com/latex.php?latex=%7B%5CPhi_r%28X%29%7D&bg=ffffff&fg=000000&s=0). From the hypothesis [(4)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt-3), we see that

![Image 371: \displaystyle  (X+a)^n = X^n + a](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%28X%2Ba%29%5En+%3D+X%5En+%2B+a&bg=ffffff&fg=000000&s=0)

in ![Image 372: {F}](https://s0.wp.com/latex.php?latex=%7BF%7D&bg=ffffff&fg=000000&s=0) for all ![Image 373: {1 \leq a \leq A}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+a+%5Cleq+A%7D&bg=ffffff&fg=000000&s=0), where ![Image 374: {A = O(r \log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7BA+%3D+O%28r+%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=000000&s=0). Note that ![Image 375: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is coprime to every integer less than ![Image 376: {A}](https://s0.wp.com/latex.php?latex=%7BA%7D&bg=ffffff&fg=000000&s=0), and thus ![Image 377: {A < p}](https://s0.wp.com/latex.php?latex=%7BA+%3C+p%7D&bg=ffffff&fg=000000&s=0).

Meanwhile, from [(2)](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#pflt) one has

![Image 378: \displaystyle  (X+a)^p = X^p + a](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%28X%2Ba%29%5Ep+%3D+X%5Ep+%2B+a&bg=ffffff&fg=000000&s=0)

in ![Image 379: {F}](https://s0.wp.com/latex.php?latex=%7BF%7D&bg=ffffff&fg=000000&s=0) for all such ![Image 380: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=000000&s=0). The two equations give

![Image 381: \displaystyle  (X^p+a)^{n/p} = (X^p)^{n/p} + a.](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%28X%5Ep%2Ba%29%5E%7Bn%2Fp%7D+%3D+%28X%5Ep%29%5E%7Bn%2Fp%7D+%2B+a.&bg=ffffff&fg=000000&s=0)

Note that the ![Image 382: {p^{th}}](https://s0.wp.com/latex.php?latex=%7Bp%5E%7Bth%7D%7D&bg=ffffff&fg=000000&s=0) power ![Image 383: {X^p}](https://s0.wp.com/latex.php?latex=%7BX%5Ep%7D&bg=ffffff&fg=000000&s=0) of a primitive ![Image 384: {r^{th}}](https://s0.wp.com/latex.php?latex=%7Br%5E%7Bth%7D%7D&bg=ffffff&fg=000000&s=0) root of unity ![Image 385: {X}](https://s0.wp.com/latex.php?latex=%7BX%7D&bg=ffffff&fg=000000&s=0) is again a primitive ![Image 386: {r^{th}}](https://s0.wp.com/latex.php?latex=%7Br%5E%7Bth%7D%7D&bg=ffffff&fg=000000&s=0) root of unity (and conversely, every primitive ![Image 387: {r^{th}}](https://s0.wp.com/latex.php?latex=%7Br%5E%7Bth%7D%7D&bg=ffffff&fg=000000&s=0) root arises in this fashion) and hence we also have

![Image 388: \displaystyle  (X+a)^{n/p} = X^{n/p} + a](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%28X%2Ba%29%5E%7Bn%2Fp%7D+%3D+X%5E%7Bn%2Fp%7D+%2B+a&bg=ffffff&fg=000000&s=0)

in ![Image 389: {F}](https://s0.wp.com/latex.php?latex=%7BF%7D&bg=ffffff&fg=000000&s=0) for all ![Image 390: {1 \leq a \leq A}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+a+%5Cleq+A%7D&bg=ffffff&fg=000000&s=0).

Inspired by this, we define a key concept: a positive integer ![Image 391: {m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0) is said to be _introspective_ if one has

![Image 392: \displaystyle  (X+a)^m = X^m + a](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle++%28X%2Ba%29%5Em+%3D+X%5Em+%2B+a&bg=ffffff&fg=000000&s=0)

in ![Image 393: {F}](https://s0.wp.com/latex.php?latex=%7BF%7D&bg=ffffff&fg=000000&s=0) for all ![Image 394: {1 \leq a \leq A}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+a+%5Cleq+A%7D&bg=ffffff&fg=000000&s=0), or equivalently if ![Image 395: {(X+a)^m=\phi_m(X+a)}](https://s0.wp.com/latex.php?latex=%7B%28X%2Ba%29%5Em%3D%5Cphi_m%28X%2Ba%29%7D&bg=ffffff&fg=000000&s=0), where ![Image 396: {\phi_m: F \rightarrow F}](https://s0.wp.com/latex.php?latex=%7B%5Cphi_m%3A+F+%5Crightarrow+F%7D&bg=ffffff&fg=000000&s=0) is the ring homomorphism that sends ![Image 397: {X}](https://s0.wp.com/latex.php?latex=%7BX%7D&bg=ffffff&fg=000000&s=0) to ![Image 398: {X^m}](https://s0.wp.com/latex.php?latex=%7BX%5Em%7D&bg=ffffff&fg=000000&s=0).

We have just shown that ![Image 399: {p, n, n/p}](https://s0.wp.com/latex.php?latex=%7Bp%2C+n%2C+n%2Fp%7D&bg=ffffff&fg=000000&s=0) are all introspective; ![Image 400: {1}](https://s0.wp.com/latex.php?latex=%7B1%7D&bg=ffffff&fg=000000&s=0) is also trivially introspective. Furthermore, if ![Image 401: {m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0) and ![Image 402: {m'}](https://s0.wp.com/latex.php?latex=%7Bm%27%7D&bg=ffffff&fg=000000&s=0) are introspective, it is not hard to see that ![Image 403: {mm'}](https://s0.wp.com/latex.php?latex=%7Bmm%27%7D&bg=ffffff&fg=000000&s=0) is also introspective. Thus we in fact have a lot of introspective integers: any number of the form ![Image 404: {p^i (n/p)^j}](https://s0.wp.com/latex.php?latex=%7Bp%5Ei+%28n%2Fp%29%5Ej%7D&bg=ffffff&fg=000000&s=0) for ![Image 405: {i,j \geq 0}](https://s0.wp.com/latex.php?latex=%7Bi%2Cj+%5Cgeq+0%7D&bg=ffffff&fg=000000&s=0) is introspective.

It turns out in fact that it is not possible to create so many different introspective numbers, basically the presence of so many polynomial identities in the field would eventually violate the [factor theorem](http://en.wikipedia.org/wiki/Factor_theorem). To see this, let ![Image 406: {{\mathcal G} \subset F^\times}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathcal+G%7D+%5Csubset+F%5E%5Ctimes%7D&bg=ffffff&fg=000000&s=0) be the multiplicative group generated by the quantities ![Image 407: {X+a}](https://s0.wp.com/latex.php?latex=%7BX%2Ba%7D&bg=ffffff&fg=000000&s=0) for ![Image 408: {1 \leq a \leq A}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+a+%5Cleq+A%7D&bg=ffffff&fg=000000&s=0). Observe that ![Image 409: {z^m = \phi_m(z)}](https://s0.wp.com/latex.php?latex=%7Bz%5Em+%3D+%5Cphi_m%28z%29%7D&bg=ffffff&fg=000000&s=0) for all ![Image 410: {z \in {\mathcal G}}](https://s0.wp.com/latex.php?latex=%7Bz+%5Cin+%7B%5Cmathcal+G%7D%7D&bg=ffffff&fg=000000&s=0). We now show that this places incompatible lower and upper bounds on ![Image 411: {{\mathcal G}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathcal+G%7D%7D&bg=ffffff&fg=000000&s=0). We begin with the lower bound:

> **Proposition 4 (Lower bound on ![Image 412: {{\mathcal G}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathcal+G%7D%7D&bg=ffffff&fg=000000&s=0))**[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/)![Image 413: {|{\mathcal G}| \geq 2^t}](https://s0.wp.com/latex.php?latex=%7B%7C%7B%5Cmathcal+G%7D%7C+%5Cgeq+2%5Et%7D&bg=ffffff&fg=000000&s=0).

_Proof:_ Let ![Image 414: {P(X)}](https://s0.wp.com/latex.php?latex=%7BP%28X%29%7D&bg=ffffff&fg=000000&s=0) be a product of less than ![Image 415: {t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0) of the quantities ![Image 416: {X+1,\ldots,X+A}](https://s0.wp.com/latex.php?latex=%7BX%2B1%2C%5Cldots%2CX%2BA%7D&bg=ffffff&fg=000000&s=0) (allowing repetitions), then ![Image 417: {P(X)}](https://s0.wp.com/latex.php?latex=%7BP%28X%29%7D&bg=ffffff&fg=000000&s=0) lies in ![Image 418: {{\mathcal G}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathcal+G%7D%7D&bg=ffffff&fg=000000&s=0). Since ![Image 419: {A \geq 2r \geq 2t}](https://s0.wp.com/latex.php?latex=%7BA+%5Cgeq+2r+%5Cgeq+2t%7D&bg=ffffff&fg=000000&s=0), there are certainly at least ![Image 420: {2^t}](https://s0.wp.com/latex.php?latex=%7B2%5Et%7D&bg=ffffff&fg=000000&s=0) ways to pick such a product. So to establish the proposition it suffices to show that all these products are distinct.

Suppose for contradiction that ![Image 421: {P(X)=Q(X)}](https://s0.wp.com/latex.php?latex=%7BP%28X%29%3DQ%28X%29%7D&bg=ffffff&fg=000000&s=0), where ![Image 422: {P,Q}](https://s0.wp.com/latex.php?latex=%7BP%2CQ%7D&bg=ffffff&fg=000000&s=0) are different products of less than ![Image 423: {t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0) of the ![Image 424: {X+1,\ldots,X+A}](https://s0.wp.com/latex.php?latex=%7BX%2B1%2C%5Cldots%2CX%2BA%7D&bg=ffffff&fg=000000&s=0). Then, for every introspective ![Image 425: {m}](https://s0.wp.com/latex.php?latex=%7Bm%7D&bg=ffffff&fg=000000&s=0), ![Image 426: {P(X^m)=Q(X^m)}](https://s0.wp.com/latex.php?latex=%7BP%28X%5Em%29%3DQ%28X%5Em%29%7D&bg=ffffff&fg=000000&s=0) as well (note that ![Image 427: {P(X^m) = \phi_m(P(X))}](https://s0.wp.com/latex.php?latex=%7BP%28X%5Em%29+%3D+%5Cphi_m%28P%28X%29%29%7D&bg=ffffff&fg=000000&s=0)). In particular, this shows that ![Image 428: {X^{m_1},\ldots,X^{m_t}}](https://s0.wp.com/latex.php?latex=%7BX%5E%7Bm_1%7D%2C%5Cldots%2CX%5E%7Bm_t%7D%7D&bg=ffffff&fg=000000&s=0) are all roots of the polynomial ![Image 429: {P-Q}](https://s0.wp.com/latex.php?latex=%7BP-Q%7D&bg=ffffff&fg=000000&s=0). But this polynomial has degree less than ![Image 430: {t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0), and the ![Image 431: {X^{m_1},\ldots,X^{m_t}}](https://s0.wp.com/latex.php?latex=%7BX%5E%7Bm_1%7D%2C%5Cldots%2CX%5E%7Bm_t%7D%7D&bg=ffffff&fg=000000&s=0) are distinct by hypothesis, and we obtain the desired contradiction by the factor theorem. ![Image 432: \Box](https://s0.wp.com/latex.php?latex=%5CBox&bg=ffffff&fg=000000&s=0)

> **Proposition 5 (Upper bound on ![Image 433: {{\mathcal G}}](https://s0.wp.com/latex.php?latex=%7B%7B%5Cmathcal+G%7D%7D&bg=ffffff&fg=000000&s=0))**[](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/) Suppose that there are exactly ![Image 434: {t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0) residue classes modulo ![Image 435: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) of the form ![Image 436: {p^i (n/p)^j \hbox{ mod } r}](https://s0.wp.com/latex.php?latex=%7Bp%5Ei+%28n%2Fp%29%5Ej+%5Chbox%7B+mod+%7D+r%7D&bg=ffffff&fg=000000&s=0) for ![Image 437: {i,j \geq 0}](https://s0.wp.com/latex.php?latex=%7Bi%2Cj+%5Cgeq+0%7D&bg=ffffff&fg=000000&s=0). Then ![Image 438: {|{\mathcal G}| \leq n^{\sqrt{t}}}](https://s0.wp.com/latex.php?latex=%7B%7C%7B%5Cmathcal+G%7D%7C+%5Cleq+n%5E%7B%5Csqrt%7Bt%7D%7D%7D&bg=ffffff&fg=000000&s=0).

_Proof:_ By the pigeonhole principle, we must have a collision

![Image 439: \displaystyle p^i (n/p)^j = p^{i'} (n/p)^{j'} \hbox{ mod } r](https://s0.wp.com/latex.php?latex=%5Cdisplaystyle+p%5Ei+%28n%2Fp%29%5Ej+%3D+p%5E%7Bi%27%7D+%28n%2Fp%29%5E%7Bj%27%7D+%5Chbox%7B+mod+%7D+r&bg=ffffff&fg=000000&s=0)

for some ![Image 440: {0 \leq i,j,i',j' \leq \sqrt{t}}](https://s0.wp.com/latex.php?latex=%7B0+%5Cleq+i%2Cj%2Ci%27%2Cj%27+%5Cleq+%5Csqrt%7Bt%7D%7D&bg=ffffff&fg=000000&s=0) with ![Image 441: {(i,j) \neq (i',j')}](https://s0.wp.com/latex.php?latex=%7B%28i%2Cj%29+%5Cneq+%28i%27%2Cj%27%29%7D&bg=ffffff&fg=000000&s=0). Setting ![Image 442: {m := p^i (n/p)^j}](https://s0.wp.com/latex.php?latex=%7Bm+%3A%3D+p%5Ei+%28n%2Fp%29%5Ej%7D&bg=ffffff&fg=000000&s=0) and ![Image 443: {m' := p^{i'} (n/p)^{j'}}](https://s0.wp.com/latex.php?latex=%7Bm%27+%3A%3D+p%5E%7Bi%27%7D+%28n%2Fp%29%5E%7Bj%27%7D%7D&bg=ffffff&fg=000000&s=0), we thus see that there are two distinct introspective numbers ![Image 444: {m,m'}](https://s0.wp.com/latex.php?latex=%7Bm%2Cm%27%7D&bg=ffffff&fg=000000&s=0) of size most ![Image 445: {n^{\sqrt{t}}}](https://s0.wp.com/latex.php?latex=%7Bn%5E%7B%5Csqrt%7Bt%7D%7D%7D&bg=ffffff&fg=000000&s=0) which are equal modulo ![Image 446: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0). (To ensure that ![Image 447: {m,m'}](https://s0.wp.com/latex.php?latex=%7Bm%2Cm%27%7D&bg=ffffff&fg=000000&s=0) are distinct, we use the hypothesis that ![Image 448: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) is not a power of ![Image 449: {p}](https://s0.wp.com/latex.php?latex=%7Bp%7D&bg=ffffff&fg=000000&s=0).) This implies that ![Image 450: {\phi_m = \phi_{m'}}](https://s0.wp.com/latex.php?latex=%7B%5Cphi_m+%3D+%5Cphi_%7Bm%27%7D%7D&bg=ffffff&fg=000000&s=0), and thus ![Image 451: {z^m = z^{m'}}](https://s0.wp.com/latex.php?latex=%7Bz%5Em+%3D+z%5E%7Bm%27%7D%7D&bg=ffffff&fg=000000&s=0) for all ![Image 452: {z \in {\mathcal G}}](https://s0.wp.com/latex.php?latex=%7Bz+%5Cin+%7B%5Cmathcal+G%7D%7D&bg=ffffff&fg=000000&s=0). But the polynomial ![Image 453: {z^m - z^{m'}}](https://s0.wp.com/latex.php?latex=%7Bz%5Em+-+z%5E%7Bm%27%7D%7D&bg=ffffff&fg=000000&s=0) has degree at most ![Image 454: {n^{\sqrt{t}}}](https://s0.wp.com/latex.php?latex=%7Bn%5E%7B%5Csqrt%7Bt%7D%7D%7D&bg=ffffff&fg=000000&s=0), and the claim now follows from the factor theorem. ![Image 455: \Box](https://s0.wp.com/latex.php?latex=%5CBox&bg=ffffff&fg=000000&s=0)

Since ![Image 456: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=000000&s=0) has order greater than ![Image 457: {\log^2 n}](https://s0.wp.com/latex.php?latex=%7B%5Clog%5E2+n%7D&bg=ffffff&fg=000000&s=0) in ![Image 458: {({\mathbb Z}/r{\mathbb Z})^\times}](https://s0.wp.com/latex.php?latex=%7B%28%7B%5Cmathbb+Z%7D%2Fr%7B%5Cmathbb+Z%7D%29%5E%5Ctimes%7D&bg=ffffff&fg=000000&s=0), we see that the number ![Image 459: {t}](https://s0.wp.com/latex.php?latex=%7Bt%7D&bg=ffffff&fg=000000&s=0) of residue classes ![Image 460: {r}](https://s0.wp.com/latex.php?latex=%7Br%7D&bg=ffffff&fg=000000&s=0) of the form ![Image 461: {p^i (n/p)^j}](https://s0.wp.com/latex.php?latex=%7Bp%5Ei+%28n%2Fp%29%5Ej%7D&bg=ffffff&fg=000000&s=0) is at least ![Image 462: {\log^2 n}](https://s0.wp.com/latex.php?latex=%7B%5Clog%5E2+n%7D&bg=ffffff&fg=000000&s=0). But then ![Image 463: {2^t > n^{\sqrt{t}}}](https://s0.wp.com/latex.php?latex=%7B2%5Et+%3E+n%5E%7B%5Csqrt%7Bt%7D%7D%7D&bg=ffffff&fg=000000&s=0), and so Propositions [4](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#lower), [5](https://terrytao.wordpress.com/2009/08/11/the-aks-primality-test/#upper) are incompatible.

_Update, Aug 12_: A thorough discussion of the AKS algorithm by Andrew Granville for Bull. Amer. Math. Soc. can be found [here](http://www.dms.umontreal.ca/~andrew/PDF/Bulletin04.pdf).

I like that this proof illustrates how fast (i.e. polynomial) computability fails to be analogous to general computability. If one has an “elementary” proof that a witness with some computably checkable property exists, the proof can generally be turned into a computation of that witness. (This can be made formal: if the proof can be formulated, even in a theory like full second order arithmetic, it can be converted to a computation of the witness.)

Yet, when given a composite number, the AKS primality test gives a fairly simple demonstration, in polynomial time, that there is some factor, but no way to quickly find what that factor is. Interestingly, the test does give a witness to _something_, and it’s the task of converting that witness into a factor which cannot be carried out quickly (in polynomial time).

“Why is it, though, that number theoretic algorithms complexity is always done in terms of the logarithm of the number sizes?”

Anonymous’s answer, that this is the size of the input, is good. One should also note that it is the efficiency of the “easy” operations: addition, multiplication, division with remainder, GCD and so forth. Since these are the building blocks of our algorithms, it makes sense to study those algorithms which are not much worse than the pieces they are built from.

![Image 488: A 32x32 small image, likely a logo, icon or avatar](https://0.gravatar.com/avatar/31e25d91ee1d87535215e41466ba8d23?s=32&d=identicon&r=PG)
Very nice explanation of the results! I started to get a bit confused about half way through with the big-O notations. For instance, you state Theorem 1 as:

> Theorem 1 (AKS theorem) Suppose that for all ![Image 489: {1 \leq a, r \leq O( \log^{O(1)} n)}](https://s0.wp.com/latex.php?latex=%7B1+%5Cleq+a%2C+r+%5Cleq+O%28+%5Clog%5E%7BO%281%29%7D+n%29%7D&bg=ffffff&fg=545454&s=0), (4) holds, and ![Image 490: {a}](https://s0.wp.com/latex.php?latex=%7Ba%7D&bg=ffffff&fg=545454&s=0) is coprime to ![Image 491: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=545454&s=0). Then ![Image 492: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=545454&s=0) is either a prime, or a power of a prime.

The first sentence looks like it can be true for all ![Image 493: n](https://s0.wp.com/latex.php?latex=n&bg=ffffff&fg=545454&s=0) while of course the second isn’t. After all, there is some constant such that (4) holds for all ![Image 494: a,r](https://s0.wp.com/latex.php?latex=a%2Cr&bg=ffffff&fg=545454&s=0) less than that constant times ![Image 495: \log n](https://s0.wp.com/latex.php?latex=%5Clog+n&bg=ffffff&fg=545454&s=0), namely 0.

I suppose the right way to understand this is that the big-O expressions are all taken to introduce existential quantifiers over constants somewhere early on, with scope over the entire theorem, rather than just scope over the specific equation the big-O statement is in. And you’re also supposed to understand the scope of this quantifier with respect to various other quantifiers appropriately too. Thus, the more formal statement of Theorem 1 must say

> ![Image 496: \exists k\forall n \textrm{If }\forall 1\leq a, r\leq k(\log^k n)](https://s0.wp.com/latex.php?latex=%5Cexists+k%5Cforall+n+%5Ctextrm%7BIf+%7D%5Cforall+1%5Cleq+a%2C+r%5Cleq+k%28%5Clog%5Ek+n%29&bg=ffffff&fg=545454&s=0) (4) holds, then ![Image 497: {n}](https://s0.wp.com/latex.php?latex=%7Bn%7D&bg=ffffff&fg=545454&s=0) is either a prime, or a power of a prime.

Switching the initial existential and universal quantifiers yields a statement that is trivially true (just choose ![Image 498: k=0](https://s0.wp.com/latex.php?latex=k%3D0&bg=ffffff&fg=545454&s=0) if ![Image 499: n](https://s0.wp.com/latex.php?latex=n&bg=ffffff&fg=545454&s=0) is prime or a power of a prime and ![Image 500: k=n](https://s0.wp.com/latex.php?latex=k%3Dn&bg=ffffff&fg=545454&s=0) otherwise).

Perhaps the fact that this existential quantifier gets widest scope is implicit in the fact that it’s called a constant, but it gets more unclear when you cross sentence boundaries. Because there are other examples with the same format as the statement of Theorem 1 here where the quantifier clearly isn’t meant to have wide scope over both sentences. For instance

> Suppose one can factor ![Image 501: n](https://s0.wp.com/latex.php?latex=n&bg=ffffff&fg=545454&s=0) in ![Image 502: n^{O(1)}](https://s0.wp.com/latex.php?latex=n%5E%7BO%281%29%7D&bg=ffffff&fg=545454&s=0) time. Then standard cryptography is weak.

This doesn’t mean that there is some exponent on the polynomial that would mean cryptography is weak – rather it means that _any_ exponent on the polynomial would work.
