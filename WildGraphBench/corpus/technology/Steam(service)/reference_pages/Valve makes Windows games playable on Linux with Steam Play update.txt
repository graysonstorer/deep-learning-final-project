Heads up developers, if your players have been asking for a Linux-compatable version of your game, Valve's announced that delivering that version should be much easier going forward.

In [a post on the Steam community forums,](https://steamcommunity.com/games/221410#announcements/detail/1696055855739350561) Valve representative Pierre-Loup Griffais announced that Valve is releasing a new version of Steam play that includes a new feature for Linux users. Using an improved version of the compatibility software Wine known as Proton, Griffais states that Linux users can now play games on Steam that are meant to run on Windows.

Additionally, Steam on Linux will support controllers supported by Steam on Windows.

While the update doesn't cover all Windows games on Steam yet, a list of games including _Doom, Star Wars: Battlefront 2, Payday: The Heist_, and more being supported in this beta, with more to come. Valve has also shared Proton's [open-source code on Github](https://steamcommunity.com/linkfilter/?url=https://github.com/ValveSoftware/Proton/) for developers interested in poking around with it themselves.
